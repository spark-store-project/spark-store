#### 调用参数(spk规则)

参数只有一个Url，该url应当遵循这种格式：`spk://<任意合法字符>/web分类/包名`

例如:

[spk://abcdefg/games/store.spark-app.hmcl](spk://abcdefg/games/store.spark-app.hmcl)


可选的web分类：

| 分类名称 | web分类   　    |
| -------- | -------------- |
| 网络应用 | network        |
| 社交沟通 | chat           |
| 音乐欣赏 | music          |
| 视频播放 | video          |
| 图形图像 | graphics       |
| 游戏娱乐 | games          |
| 办公学习 | office         |
| 阅读翻译 | reading        |
| 编程开发 | development    |
| 系统工具 | tools          |
| 主题美化 | beautify       |
| 其他应用 | others         |
